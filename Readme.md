###How to start using it###

- Take the source

```
$ cd ~
$ git clone http://github.com/zlatozar/pecan .emacs.d
```

- Download bootstrap packages

```
$ cd ~/.emacs.d/site-lisp
$ make
```
- Run Emacs and all additional packages will be downloaded from MELPA
